
# 🛒 E-Commerce Website Clone

A responsive e-commerce website clone built to practice modern front-end web development concepts. This project recreates the user interface and core shopping experience of a typical online store using **HTML**, **CSS**, and **JavaScript**. The focus was on building a clean, responsive, and user-friendly interface while understanding how different components of an e-commerce platform work together.

The project includes a modern homepage, product listings, category sections, a shopping cart interface, product search functionality, and interactive UI elements. It is designed to provide a smooth browsing experience across desktop, tablet, and mobile devices.

## ✨ Features

* Responsive design for all screen sizes
* Modern and clean UI
* Product listing section
* Product cards with images, prices, and ratings
* Product search functionality
* Category-based product display
* Shopping cart functionality
* Add to Cart and Remove from Cart
* Quantity update controls
* Dynamic cart total calculation
* Navigation bar with mobile support
* Interactive buttons and hover effects
* Smooth scrolling and UI interactions
* Organized project structure
* Fast-loading static website

## 🛠️ Technologies Used

* HTML5
* CSS3
